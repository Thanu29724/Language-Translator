# 🌍 Language Translator

A powerful and user-friendly **Language Translator Application** built using Python that enables seamless communication across the system. This project supports real-time text translation along with advanced features like speech input and text-to-speech output.

---

## 🚀 Features

- 🌐 Translate text across 100+ languages
- 🔍 Automatic language detection
- 🔊 Text-to-Speech (Read Aloud)
- 📋 Copy translated text instantly
- 🧹 Clear input/output easily
- 🖥️ Simple and intuitive GUI

---

## 🛠️ Tech Stack

- Python
- Tkinter (GUI)
- Googletrans (Translation API)

---

## 📦 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Thanu29724/Language-Translator.git
   ```

2. Navigate to the project folder:
   ```bash
   cd Language-Translator
   ```

3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

   *(If requirements.txt is not available, install manually:)*  
   ```bash
   pip install googletrans gtts SpeechRecognition pyperclip tkinter
   ```

---

## ▶️ Usage

Run the application:

```bash
python main.py
```

---

## 💡 How It Works

- The app detects the input language automatically.
- Uses **Google Translate API** to convert text into the selected language.
- Converts translated text into audio using **gTTS**.
- Supports voice input via **SpeechRecognition** for easy translation. :contentReference[oaicite:1]{index=1}
